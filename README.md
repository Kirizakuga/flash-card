<h3> in case u have no idea how to install and run this local in your machine</h3>
<p>this is the installation guide</p>
<br>
<h3>Step 1:</h3>
<p>git clone https://github.com/Kirizakuga/flash-card.git</p>
<p> or jst download the release version </p>
<br>
<h3>Step 2:</h3>
<p>Windows:	python -m venv venv</p>
<p>.\venv\Scripts\activate </p>
<br>
<p>macOS:	python3 -m venv venv</p>
<p>source venv/bin/activate</p>
<br> 
<h3>Step 3: Install requierments</h3>
<p>pip install -r requirements.txt</p>
<br>
<h3>now u can run the proj by app.py or run it by click some button</h3>
<h2>Important:</h2>
<p>if you have any issue when running the proj locally, it may be caused by the mismatch of the lib version. You shall go to the <strong>requirements.txt</strong> and update all the date of those lib manually and then do the pip install again. From now everythings should work properly, if not, you can contact the author or just fix the problem by yourself ( i personally recommend the last option )</p>
